loads.comstock

COMstock data accessor

The COMstock class is a Pandas data frame loaded with the COMstock building data.

Commercial building types are coded using three characters C, {'L','M','S'} (large, medium, and small), and {'F','H','L','O','E','R','W'} (food, health, lodging, office, education, retail, and warehouse). Load values are given in W/sf.

Example:

To get the large office load data for Alameda CA use the command

print(COMstock(state="CA",building_type="CLO"))

114    def __init__(self,
115        state:str,
116        county:str=None,
117        building_type:list[str]=None,
118        freq:str|None="1h",
119        ):
120        """Construct a COMstock data frame
121
122        Arguments:
123
124        - `state`: specifies the state (e.g., "CA")
125
126        - `county`: specifies the county (e.g., "Alameda") or None for the
127          entire state
128
129        - `building_type`: specifies the building type (e.g., "house")
130
131        - `freq`: specifies the sampling interval (None for raw sampling)
132        """
133        assert building_type in self.BUILDING_TYPES, \
134            f"{building_type=} is not one of {self.BUILDING_TYPES}"
135
136        if self.CACHEDIR is None:
137            self.CACHEDIR = os.path.join(os.path.dirname(__file__),".cache")
138        os.makedirs(self.CACHEDIR,exist_ok=True)
139
140        # check cache
141        file = f"{state}_{building_type}.csv.gz" \
142            if county is None \
143            else f"{state}_{county}_{building_type}.csv.gz"
144        cache = os.path.join(self.CACHEDIR,file.replace(" ","-"))
145        if not os.path.exists(cache):
146
147            # download data to cache
148            root = "https://oedi-data-lake.s3.amazonaws.com/nrel-pds-building-stock/" \
149                "end-use-load-profiles-for-us-building-stock/2021/" \
150                "comstock_amy2018_release_1/timeseries_aggregates"
151            btype = self.BUILDING_TYPES[building_type]
152            if county is None:
153                url = f"{root}/by_state/state={state.upper()}/{state.lower()}-{btype}.csv"
154            else:
155                fips = County(ST=state,COUNTY=county).FIPS
156                url = f"{root}/by_county/state={state.upper()}/g{fips[:2]}0{fips[2:]}0-{btype}.csv"
157            try:
158                data = pd.read_csv(url)
159            except urllib.error.HTTPError as err:
160
161                # download error (most likely no data in COMstock)
162                warnings.warn(f"COMstock building type '{btype}' has no data ({err})")
163
164                # create all zeros dataframe
165                ndx = pd.date_range(
166                    start="2018-01-01 05:00:00+00:00",
167                    end="2019-01-01 04:00:00+00:00",
168                    freq=freq)
169                zeros = [0.0]*len(ndx)
170                data = pd.DataFrame(data={x:zeros for x in self.COLUMNS},index=ndx)
171                data.index.name = "timestamp"
172                data.reset_index(inplace=True)
173                data["floor_area_represented"] = 0.0
174
175            data.to_csv(cache,compression="gzip" if cache.endswith(".gz") else None)
176
177        # load data from cache
178        data = pd.read_csv(cache,dtype=str,na_filter=False,low_memory=False)
179        data.set_index(["timestamp"],inplace=True)
180        data.index = (pd.DatetimeIndex(data.index,tz=pytz.timezone("EST")) \
181            - dt.timedelta(minutes=15)).tz_convert(pytz.UTC)
182
183        # capture number floor_area
184        floor_area = data["floor_area_represented"].astype(float)
185        if floor_area.min() != floor_area.max():
186            warnings.warn(f"{state=} {county=} floor area changes (using max)")
187        floor_area = floor_area.max()
188
189        # restructure data
190        data.drop([x for x in data.columns if x not in self.COLUMNS],inplace=True,axis=1)
191        data.rename(self.COLUMNS,inplace=True,axis=1)
192        for value in self.COLUMNS.values():
193            data[value] = [0.0 if floor_area == 0.0 else _float(x)/floor_area*1000 for x in data[value]]
194
195        data["floor_area"] = floor_area
196
197        # resample if necessary
198        if not freq is None:
199            ts = data.index.diff().mean().total_seconds()/3600
200            data = (data/ts).resample(freq).ffill()
201
202        # move year-end data to beginning
203        data.index = pd.DatetimeIndex([str(x).replace("2019","2018") for x in data.index])
204        super().__init__(data.sort_index())

Construct a COMstock data frame

The data frame includes the columns specified by COLUMNS constant, which maps the COMstock data to the data frame columns. The values are given in average Watts per unit floor area in square feet. The number of units from COMstock is given by the floor_area column. Note that the floor area is that used in the COMstock model, which may not be accurately reflect the actual floor area in any given year.
